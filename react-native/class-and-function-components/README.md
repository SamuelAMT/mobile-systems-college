# Text Generator App

A React Native application built with Expo and React Native Paper that demonstrates both functional and class-based component implementations.

## Features
- Text input field for name entry
- Text generation based on user input
- Clean Material Design UI using React Native Paper
- Implementation in both functional and class components

## Technologies Used
- React Native
- Expo
- React Native Paper

## Project Structure
```
├── function/
│   └── functionApp.js    # Functional component implementation
├── class/
│   └── classApp.js       # Class component implementation
└── App.js                # Main application file
```

## How to Run
1. Open the project in Expo Snack
2. Dependencies will be automatically installed
3. Run the application in the Expo Go app or in a web browser

## Learning Objectives
- Understanding React Native basics
- Comparing functional and class-based components
- Working with UI libraries (React Native Paper)
- Form handling in React Native
